Web Application Resources

This page contains useful resources that help me learn and grow as a frontend and UX-focused web developer. These resources work as a personal cheat sheet for building modern, user-friendly web applications.

1. Will AI Replace UX Designers?

UX Design Institute – Will AI Replace UX Designers?

This article explains how artificial intelligence is growing in the design industry, but it is very unlikely to replace UX designers. UX design depends on human skills like empathy, understanding user behavior, and collaboration. The article also shows how AI can help designers by improving efficiency, accessibility, and data analysis.

This resource is useful because it helps reduce fear about AI replacing design jobs. I will use it to understand how AI can support my work while keeping user-centered design as a priority.

Usefulness rating: 5 out of 5

2. What’s Behind All These Tech Layoffs?

Built In – What’s Behind All These Tech Layoffs?

This article explains why many tech companies are laying off workers in 2025. It shows that layoffs are mostly caused by over-hiring during the pandemic, slower growth, economic pressure, and companies shifting budgets toward AI. The article explains that this is more of a reset than a collapse of the tech industry.

This resource helps me understand the current tech job market and what skills are important to stay competitive. I will use it to stay informed and focus on learning in-demand skills.

Usefulness rating: 4.5 out of 5

3. The Frontend Developer Roadmap (2025/2026)

DEV Community – Frontend Developer Roadmap

This roadmap explains the skills, tools, and mindset needed to become a competitive frontend developer. It covers HTML, CSS, JavaScript, frameworks, accessibility, performance, testing, and professional workflows. It also shows how skills grow from beginner to senior level.

This resource is useful because it organizes a complex field into a clear learning path. I will use it to guide my studies and make sure I am building strong frontend foundations.

Usefulness rating: 5 out of 5

4. MDN Web Docs

MDN Web Docs – Mozilla Developer Network

MDN Web Docs is a trusted reference for HTML, CSS, and JavaScript. It provides clear explanations, examples, and best practices written by Mozilla and the developer community. MDN also documents web APIs and browser-based services that are commonly used in cloud-connected applications, such as Fetch API and authentication flows.

This resource is useful because it helps me understand how web technologies work and how to fix issues in my code. I will use MDN as my main technical reference during development.

Usefulness rating: 5 out of 5

5. Nielsen Norman Group (NN/g)

Nielsen Norman Group – UX Research & Usability

Nielsen Norman Group is a UX research website that shares articles about usability, accessibility, and user behavior. The content is based on real research and explains how people interact with digital products.

This resource is useful because it helps me make better design decisions based on user research. I will use it to improve usability and accessibility in my projects.

Usefulness rating: 4.5 out of 5
